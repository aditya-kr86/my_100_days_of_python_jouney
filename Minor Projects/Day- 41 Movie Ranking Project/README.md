# 🎬 Movie Ranking Project

A simple HTML project to display and rank movies!

## 📜 Description

This project is a basic HTML page that allows users to view and rank their favorite movies.

### Example:
```
The HTML page displays a list of movies with their respective ranks and allows users to update the rankings.
```

## 🚀 Getting Started

### Prerequisites

- A web browser to view the HTML file

### How to Run

1. Clone this repository:
   ```bash
   git clone --depth 1 --filter=blob:none --sparse https://github.com/aditya-kr86/my_100_days_of_python_jouney.git
   cd my_100_days_of_python_jouney
   git sparse-checkout set "Minor Projects/Day-41 Movie Ranking Project"
   ```
2. Navigate to the Minor Projects directory:
   ```bash
   cd Minor\ Projects/
   ```
3. Navigate to the Day-41 Movie Ranking Project directory:
   ```bash
   cd Day-41\ Movie\ Ranking\ Project/
   ```
4. Open the `movie_ranking.html` file in a web browser:
   ```bash
   open movie_ranking.html
   ```

## 📦 Features
